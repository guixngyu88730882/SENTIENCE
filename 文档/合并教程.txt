# AI NPCS / Sentience V4C 超详细安装指南

## 一句话说明

这是一个给 **GTA V** 用的 AI NPC Mod。装好以后，游戏里的 NPC 不再是木头人，而是可以和你打字、语音聊天，还会有性格、记忆、情绪反应。项目现在带有可视化安装器 `AI_NPCS.exe`，支持自动检测 GTA V、检测环境、一键安装和完整测试。

---

# 先看结论：你到底要装哪些东西？

你可以把整个安装过程理解成 **4 个大件 + 1 个配置文件**：

**第 1 件：GTA V 本体**
你得先有 GTA V，并且知道它装在哪个文件夹。

**第 2 件：Python**
给语音功能和一些依赖用的。安装时最重要的是：**必须勾选 “Add Python to PATH”**。

**第 3 件：LM Studio 或云端 API**
这是 NPC 的“大脑”。

* 不想花钱：用 **LM Studio 本地模型**
* 想省事：用 **DeepSeek / OpenAI 这类云 API** 

**第 4 件：ScriptHookV + ScriptHookVDotNet**
这是 GTA V 跑模组的基础框架。没有它，Mod 基本跑不起来。

**第 5 件：Mod 文件本体**
把 `sentience.dll / Newtonsoft.Json.dll / NAudio.dll` 之类文件丢进 `scripts` 文件夹。

**最后还有 1 个配置文件：`config.ini`**
第一次运行后会自动生成在：
`我的文档\GTA5MOD2026\config.ini`
不是游戏目录，也不是 scripts 文件夹。这个位置一定别放错。 

---

# 这套东西能干什么？

目前资料里写明的功能包括：

* 自动检测 GTA V 安装目录
* 一键安装核心 MOD 文件
* 自动检测环境：Python、依赖包、Whisper、ScriptHook、资源完整性
* 启动/停止语音服务
* 互动菜单
* 支持文字对话和麦克风语音输入
* NPC 记忆、性格、语音、情绪
* 新版本里还加了更强的配置系统、增强记忆、Awakening/“觉醒”系统、AI 控制 NPC 等功能。

# 你的电脑能不能跑？

最低要求：

* CPU：Intel i5-7400 / Ryzen 3 1200
* 显卡：任意支持 DirectX 11 的显卡
* 内存：8GB
* 硬盘：10GB 可用空间

推荐配置：

* CPU：i5-12400 / i5-13400 / Ryzen 5 5600X
* 显卡：GTX 1060 6GB
* 内存：32GB
* 系统：Win10 / Win11 都可以

作者还特别说明：开发测试机甚至是 **GT730 + i5-10400**。所以理论上中低配也不是完全不能玩，只是模型要选小一点。

# 最适合小白的安装路线

## 推荐直接走这个顺序，不容易翻车：

1. 安装 Python
2. 安装 LM Studio
3. 下载并加载模型
4. 打开 LM Studio 本地服务器
5. 安装 ScriptHookV
6. 安装 ScriptHookVDotNet
7. 复制 Mod 文件到 `scripts`
8. 一键安装 Python 依赖
9. 安装 Whisper 语音识别模型
10. 跑一键测试
11. 进游戏验证

# 第 0 步：先找到 GTA V 根目录

所谓 **GTA V 根目录**，就是 `GTA5.exe` 所在的那个文件夹。
常见位置：

* Steam：`...\Steam\steamapps\common\Grand Theft Auto V\`
* Epic：`...\Epic Games\GTAV\`

总之你只要记住：**能看到 `GTA5.exe` 的那个文件夹，就是根目录。** 

---

# 第 1 步：安装 Python

## 这是最容易翻车的一步

1. 打开 Python 官网下载安装包
2. 运行安装程序
3. **最关键：勾选 `Add Python to PATH`**
4. 点击 `Install Now`
5. 装完以后按 `Win + R`，输入 `cmd`
6. 输入：

```bash
python --version
```

如果能显示 `Python 3.x.x`，就说明成功。

## 小白注意

**不勾 PATH，后面很多功能都会报错。**
你可以把它理解成：
“告诉 Windows，Python 装在哪儿了。”

如果装完输入 `python --version` 没反应，十有八九就是这里没勾。

---

# 第 2 步：安装 LM Studio

## 这是 AI 大脑

1. 打开 LM Studio 官网并安装
2. 打开软件后，建议先把界面语言切成中文
3. 找到 **本地 LLM 服务 / Local Server** 相关页面
4. 准备下载模型

---

# 第 3 步：下载模型

## 推荐模型：`qwen2.5-3b-instruct`

教程里推荐在 LM Studio 搜索：

* `qwen2.5-3b-instruct`
* 选择 `Q4_K_M` 版本
* 下载大小大约 1~2GB

## 电脑性能怎么选模型？

配置教程给了很清楚的建议：

* **弱鸡电脑 / 核显 / 很弱显卡**：`qwen2.5-1.5b-instruct`
* **一般电脑**：`qwen2.5-3b-instruct`
* **强一点的电脑**：`qwen2.5-7b-instruct`

## 直接抄作业

* 电脑一般，直接上 `qwen2.5-3b-instruct`
* 电脑很烂，别硬上 7B
* 越大不一定越爽，可能只是更卡

---

# 第 4 步：加载模型并启动本地服务器

下载完模型后：

1. 在 LM Studio 里选中你刚下载的模型
2. 点击 `Load Model`
3. 切到左边的 `Local Server`
4. 把 `Start stopped` 打开，直到变成 **`Server running`**
5. 看到 `http://127.0.0.1:1234` 这类地址，说明服务起来了

## 每次开游戏前都要做什么？

教程写得很直接：

* 打开 LM Studio
* 加载模型
* 启动服务器
* 不要把它关掉

因为 **LM Studio 就是 NPC 的大脑，不开它 NPC 没法思考。**  

---

# 第 5 步：把 LM Studio 设置对

## 这一段特别重要，不然会出现奇怪输出

在服务器页面右边的 Inference 设置里：

* **System Prompt**：留空
* **Context Length**：设成 `2048`
* **Structured Output**：关闭
* **Stop Strings**：按教程加一个换行停止项
* **Temperature**：不用管，Mod 会自己处理

## 为什么这样配？

因为这个 Mod 自己会发系统提示词、自己控制格式和温度。
你在 LM Studio 里乱填，反而可能把输出搞坏。

---

# 第 6 步：安装 ScriptHookV 和 ScriptHookVDotNet

## 这是 GTA Mod 的地基

进入你资源包里的前置工具目录，找到：

* `ScriptHookV_x.x.x.zip`
* `ScriptHookVDotNet_x.x.zip`

然后做这几件事：

### 先装 ScriptHookV

把下面两个文件复制到 **GTA V 根目录**（也就是 `GTA5.exe` 同级）：

* `ScriptHookV.dll`
* `dinput8.dll`

### 再装 ScriptHookVDotNet

同样复制到 **GTA V 根目录**：

* `ScriptHookVDotNet.asi`
* `ScriptHookVDotNet2.dll`
* `ScriptHookVDotNet3.dll`

## 记住

**这几个文件不是放进 scripts 文件夹，是放到 GTA 根目录。**

---

# 第 7 步：安装 AI Mod 本体

## 这一步就是把真正的 DLL 放进去

1. 进入 GTA V 根目录
2. 找到 `scripts` 文件夹
3. 如果没有，就手动新建一个，名字必须叫：`scripts`
4. 打开资源包里的对应目录
5. 把这些文件复制进去：

* `sentience.dll`
* `Newtonsoft.Json.dll`
* `NAudio.dll`
* 以及其它配套文件

## 一句话记忆

* **根目录**：放 ScriptHook 那一堆
* **scripts 文件夹**：放 sentience.dll 这一堆

---

# 第 8 步：一键安装 Python 依赖

## 懒人专用

直接双击：

* `一键安装依赖.bat`
* 或 `install_all.bat`

它会自动安装语音识别和语音合成相关依赖。看到“安装完成”就行。

---

# 第 9 步：安装 Whisper 语音识别模型

## 不装这个，J 键语音可能没反应

有两种方法。

### 方法 A：复制现成模型

把整个 `Whisper_Model` 文件夹复制到：

`C:\whisper-tiny\`

最后要确保这个目录下有：

* `config.json`
* `model.bin`
* `tokenizer.json`
* `vocabulary.txt`

### 方法 B：在线下载

直接双击：

* `download_whisper.bat`

大概 75MB。

# config.ini 是干什么的？

## 它就是你的总开关

这个项目从 v3.1 开始，配置系统已经比较完整，支持：

* `[LLM]`：大模型设置
* `[Performance]`：性能参数
* `[TTS]`：语音设置
* `[Awakening]`：觉醒系统相关设置

---

# 最重要的配置：本地模型 和 云 API 怎么切？

## 这块最容易把人搞晕

配置教程已经说透了：

**只填 API Key 不够，必须改 `Provider`。**
如果你不改 `Provider`，程序还是会继续走本地 LM Studio。

## 情况 A：你想用本地 LM Studio，免费

把 `[LLM]` 配成这样：

```ini
[LLM]
Provider=local
LocalEndpoint=http://127.0.0.1:1234/v1/chat/completions
LocalModel=qwen2.5-3b-instruct
```

## 情况 B：你想用 DeepSeek

```ini
[LLM]
Provider=cloud
CloudEndpoint=https://api.deepseek.com/v1/chat/completions
CloudModel=deepseek-chat
CloudAPIKey=你的key
```

## 情况 C：你想用 OpenAI

```ini
[LLM]
Provider=cloud
CloudEndpoint=https://api.openai.com/v1/chat/completions
CloudModel=gpt-4o-mini
CloudAPIKey=你的key
```

这些写法都在配置教程和示例配置里明确给了。

# 一个非常重要的安全提醒

你上传的示例 `config.ini` 里包含一串看起来像真实的 `CloudAPIKey`。这类密钥不应该公开传播，建议你立刻去对应平台**撤销并重新生成**。我这里不重复展示那串 key。来源文件里确实存在该字段。

# 默认配置大概是什么意思？

示例配置里还给了这些参数：

* `MaxTokens=80`：回答别太长，速度更快
* `Temperature=0.7`：有一点创造性
* `MaxDialogueLength=45`：中文对话长度控制
* `RequestCooldown=2`：请求冷却时间
* `StrictMode=True`：更偏快响应
* `TTSProvider=edge`：默认用 Edge 语音
* `VoiceEnabled=True`：语音开启
* `Awakening Enabled=True`：觉醒系统开启

对小白来说，**默认先别动太多**。先能跑，再慢慢调。

# 游戏里怎么用？

装好进游戏后，靠近 NPC 会出现互动菜单：

* `G`：夸奖 NPC
* `H`：骂 NPC
* `T`：打字对话
* `J`：语音对话
* `F6`：查看 NPC 状态
* `F8`：开关 NPC 语音
* `F9`：保存 NPC 记忆

# NPC 现在有哪些特性？

资料里写的性格包括：

* Kind（友善）
* Angry（暴躁）
* Timid（胆小）
* Funny（搞笑）
* Cold（冷漠）

不同性格会影响互动反应。比如 Angry 可能会追你，Timid 会害怕。

# 常见翻车点

## 1）按 G / H / T / J 没反应

先查这三个：

* LM Studio 有没有开
* 你是不是离 NPC 太远
* `scripts` 文件夹里 DLL 有没有放对

## 2）NPC 有字幕没声音

查这三个：

* F8 有没有把语音关掉
* 系统音量是不是太低
* Python 和 edge-tts 有没有装好

## 3）J 键语音没反应

查这三个：

* Windows 麦克风权限有没有开
* `C:\whisper-tiny\` 里模型在不在
* 一键测试是不是通过

## 4）NPC 头上是方块字

这是已知现象，不算你装错。
因为 GTA V 的 3D 文字不支持中文，中文会显示到右下角通知区。更新日志和 FAQ 都有提到。 

## 5）游戏卡

* LM Studio 不要同时加载太多模型
* 关掉别的吃 CPU 的程序
* 显卡弱就换小模型
