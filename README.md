## 🌟 特性
- **🤖 100% 本地 AI**: 无需联网，无需 API 费用。使用 LM Studio 驱动。
- **🗣️ 全双工语音**: 听得懂你说的话 (Whisper)，也能用语音回答你 (TTS)。
- **🧠 长期记忆**: NPC 记得你昨天做过什么，记得你的名字。

- **🔒 隐私优先**: 所有数据留在你的电脑里。

## 🚀 快速开始
1.语言模型
   ─────────────────
   a. 在LM Studio搜索栏输入：qwen2.5-1.5b-instruct
   b. 选择 Q5_K_M 版本
   c. 点击下载（约1.1GB）
   详情查看https://www.bilibili.com/video/BV1VBASzHE6r/?vd_source=faf111cb517a8b03439accdd05a06a5d#reply116141519344369
─────────────────────────────────────────
2.音频模型

方法A：使用我们提供的模型（推荐）
   a. 创建文件夹：C:\whisper-tiny\
   b. 将整个 Whisper_Model 文件夹
   （音频模型在此处获取https://www.wanjiadongli.com/mods/295861）
      复制到 C:\whisper-tiny\
   c. 确保路径为：
      C:\whisper-tiny\config.json
      C:\whisper-tiny\model.bin
      C:\whisper-tiny\tokenizer.json
      C:\whisper-tiny\vocabulary.txt

**确保Python 3.10+ 已安装并添加到环境变量*
